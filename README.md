# ANGELS-Mascon
The **AN**alyst of **G**ravity **E**stimation with **L**ow-orbit **S**atellites:
The **Mas**s **con**centration
## Main function
ANGELS-Mascon is a software that enables the user to process the gravity satellite data in the Mascon approch.
